# Админка

## Обновление файла зависимостей

Для админки обновляем файл `admin/admin-pyproject.toml`

## Сбор файла зависимостей админки

Чтобы собрать файл `pyproject.toml` для сервиса админки необходимо:
- из корня проекта активировать окружение
```bash
source .venv/bin/activate
```
- запустить команду `make admin-dependencies-file`
